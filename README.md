# Labyrinth Algorithm Generator  

A simple Java-based program for generating and visualizing labyrinths using a dfs algorithm.

---

## How to Run the Code

### **Requirements**

* Java 8 or higher

---

## Execution

Run the main program using:

```
java -jar LabyrinthApp.jar [width] [height] [size_of_shapes] [type_of_shapes]
```

### **Available Shape Types**

* TRIANGLE
* SQUARE
* HEXAGON

---

### **Example**

```
java -jar LabyrinthApp.jar 40 40 10 SQUARE
```

This runs the algorithm with:

* Width: 40
* Height: 40
* Shape size: 10
* Shape type: Square
